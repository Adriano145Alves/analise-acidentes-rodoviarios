\# Coleta de Dados



Os dados utilizados neste projeto são provenientes de bases públicas de registros de acidentes rodoviários.



Esses dados incluem informações como:



\- Data do acidente

\- Estado (UF)

\- Município

\- Tipo de acidente

\- Causa do acidente

\- Número de mortos

\- Número de feridos

\- Índice de gravidade



Os dados foram organizados em formato estruturado utilizando planilhas para permitir análise e visualização.
