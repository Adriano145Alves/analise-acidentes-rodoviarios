\# Análise de Acidentes Rodoviários no Brasil



Este projeto tem como objetivo analisar padrões de acidentes rodoviários no Brasil utilizando análise de dados e visualização em dashboard.



\## Ferramentas utilizadas



\- Google Planilhas

\- Looker Studio

\- GitHub



\## Dashboard



O dashboard interativo pode ser acessado no link abaixo:



https://lookerstudio.google.com/reporting/ea46eb1f-05e6-4b88-9a71-5279d84d0233



\## Objetivo



Identificar padrões temporais, geográficos e causas associadas aos acidentes rodoviários para auxiliar na compreensão do problema.
